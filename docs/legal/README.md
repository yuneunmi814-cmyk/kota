### 약관 호스팅 (Google Play 개인정보처리방침 URL + 앱 약관 링크)
`site/` 에 위 4개 문서를 정적 HTML로 변환해 둠(`terms/ privacy/ location/ refund/` + `index.html`).
- **재생성**: `node /tmp/build_legal_site.js` (마크다운 수정 후 다시 실행 — `marked` 사용)
- **호스팅**: GitHub Pages(예: `docs/` 또는 별도 레포) / Netlify에 `site/` 업로드 → 공개 URL 확보
- 앱 연결: `mobile/.env`의 `EXPO_PUBLIC_LEGAL_BASE_URL`을 그 베이스로 설정하면 `AboutScreen`의 약관 링크가 `/terms /privacy /location /refund`로 연결됨
- ⚠️ 게시 전 `[ ]` 빈칸(상호·CPO·연락처 등)을 채우고 변호사 검토 필수

## 출시 전 법적 절차 체크 (사업자 본인 수행) — 상세는 [출시_인허가_가이드.md](출시_인허가_가이드.md)

1. ✅ **사업자등록 완료** (프로젝트윤 / 간이과세자)
2. **위치기반서비스사업 신고** (방통위, 소상공인등 간소화) — 가이드 모드 출시 **선행조건**, 무료
3. **통신판매업 신고** — ⚠️ **간이과세자라 현재 면제 대상**. 일반과세 전환·거래 증가 시 신고(이때 에스크로 이용확인증 필요). 단 **통신판매중개자 고지·신원표시는 지금도 적용**
4. **개인정보 보호책임자(CPO) 지정** + 처리방침 게시
5. **구글 플레이** — Console 계정($25)·Data safety·콘텐츠 등급. 유료화 시 **인앱결제 정책** 확정(가이드 3장)
6. (유료화 시) **PG 가맹** + **크리에이터 정산 세무**(비사업자 원천징수 3.3%)

## 데이터 처리 현황 (처리방침 작성 근거 — 실제 코드 기준)

- 수집: 이메일·닉네임·비밀번호(bcrypt)·소셜 식별자, 동의 이력, (가이드 모드)위치좌표, 푸시 토큰, 리뷰/북마크/여행 기록, (크리에이터)정산 계좌 — *정산계좌는 별도 동의·암호화 예정*
- 보관/파기: 탈퇴 시 soft delete → **30일 후 완전 파기**(배치), 체크인 좌표 **6개월 후 NULL**, 동의 이력은 불변 보존
- 위탁/제공: PG사(결제), 클라우드(AWS), 푸시(FCM/Google), 지도(Kakao) — 처리방침에 수탁자 명시 필요
